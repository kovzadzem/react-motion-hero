# react-motion-hero
# React Motion Hero - პროექტის აღწერა

## პროექტის შესახებ

ეს პროექტი წარმოადგენს თანამედროვე ანიმაციურ Hero Section-ს, რომელიც შექმნილია React-ის გამოყენებით.

მთავარი მიზანი იყო შემექმნა ინტერაქტიული და ვიზუალურად მიმზიდველი მთავარი სექცია, სადაც გამოყენებულია თანამედროვე ვებ-დიზაინის ტექნიკები, როგორიცაა Glassmorphism, 3D მოძრაობები და დინამიკური ანიმაციები.

პროექტში მომხმარებელს შეუძლია ურთიერთქმედება ელემენტებთან:
- მაუსის მოძრაობით იცვლება 3D პერსპექტივა
- ბარათები მოძრაობენ და რეაგირებენ მოქმედებაზე
- დაჭერისას დამატებითი ინფორმაცია იხსნება
- გამოიყენება განათების და ჩრდილების ეფექტები

---

# გამოყენებული ტექნოლოგიები

## React

React გამოვიყენე კომპონენტების შესაქმნელად და პროექტის სტრუქტურის დასალაგებლად.

პროექტი დაყოფილია დამოუკიდებელ კომპონენტებად, რაც კოდის გამარტივებას და ხელახლა გამოყენებას უზრუნველყოფს.

გამოყენებული კომპონენტები:

- HeroVisual.jsx
- FloatingCard.jsx


---

## Vite

Vite გამოყენებულია პროექტის დასაწყებად და სწრაფი განვითარების გარემოს შესაქმნელად.

მისი დახმარებით პროექტი სწრაფად იტვირთება და ცვლილებები რეალურ დროში ჩანს.


---

## Tailwind CSS

Tailwind CSS გამოვიყენე დიზაინის შესაქმნელად.

მისი საშუალებით გაკეთებულია:

- Responsive Layout
- Glassmorphism ეფექტები
- ფერები
- ზომები
- დაშორებები
- ბარათების დიზაინი


---

## Framer Motion

Framer Motion გამოყენებულია ყველა ანიმაციისთვის.

მისი დახმარებით განხორციელდა:

- ელემენტების გამოჩენის ანიმაცია
- Floating მოძრაობები
- Hover ეფექტები
- 3D Rotation
- Smooth Transition-ები


---

# პროექტის სტრუქტურა

## HeroVisual.jsx

ეს არის მთავარი კომპონენტი.

მისი პასუხისმგებლობებია:

- მთავარი Glass Card-ის შექმნა
- Mouse Parallax ეფექტის მართვა
- Background Glow ეფექტები
- პატარა ბარათების განთავსება


---

## FloatingCard.jsx

ეს არის განმეორებადი კომპონენტი, რომელიც გამოიყენება პატარა ბარათებისთვის.

თითოეულ ბარათს აქვს:

- საკუთარი სათაური
- მონაცემი
- ანიმაცია
- Mouse Glow ეფექტი
- Click Interaction


---

# ანიმაციების მუშაობის პრინციპი

## Mouse Parallax

მაუსის პოზიციის მიხედვით იცვლება მთავარი ბარათის:

- X ღერძის მოძრაობა
- Y ღერძის მოძრაობა
- Rotation

ეს ქმნის 3D სივრცის ეფექტს.


---

## Floating Animation

პატარა ბარათები მუდმივად მოძრაობენ ზემოთ და ქვემოთ.

ეს ქმნის მცურავი ობიექტების ეფექტს და დიზაინს უფრო ცოცხალს ხდის.


---

## Interactive Cards

ბარათზე დაჭერისას:

- ბარათი მოძრაობს ზემოთ
- ჩნდება დამატებითი დიზაინი
- აქტიურდება განათების ეფექტი


---

# Responsive დიზაინი

პროექტი ადაპტირებულია სხვადასხვა მოწყობილობისთვის:

- Desktop
- Laptop
- Tablet
- Mobile


Responsive დიზაინისთვის გამოყენებულია Tailwind CSS-ის breakpoint სისტემა:

- sm
- md
- lg


ეკრანის ზომის ცვლილებისას ავტომატურად იცვლება:

- ბარათების ზომები
- პოზიციები
- დაშორებები
- ანიმაციების ქცევა


---

# პროექტის მიზანი

ამ პროექტის მიზანი იყო თანამედროვე Frontend ტექნოლოგიების გამოყენებით შემექმნა პროფესიონალური Hero Section, რომელიც აერთიანებს:

- თანამედროვე დიზაინს
- მომხმარებელთან ინტერაქციას
- ანიმაციებს
- კომპონენტურ სტრუქტურას
- Responsive მიდგომას


---

# დასკვნა

პროექტის შექმნისას გამოყენებული იქნა React-ის კომპონენტური მიდგომა და Framer Motion-ის ანიმაციური შესაძლებლობები.

შედეგად მიღებულია თანამედროვე, ინტერაქტიული და მრავალმოწყობილობაზე მორგებული Hero Section.
